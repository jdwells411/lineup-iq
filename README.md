# LineupIQ

AI-powered Little League lineup generator. Automatically creates rule-compliant game lineups based on player skills, coaching strategy, and league rules.

## Features (Planned)
- Player roster with 10-attribute skill ratings
- Season schedule management (opponent, date, time, location, home/away)
- Configurable rules engine (infield minimums, bench fairness, pitching requirements)
- Coaching strategy rules (position pairing logic)
- Per-game pitcher/catcher planning with advisory warnings
- One-click lineup generation
- Season-level stat tracking across games
- Visual grid + diamond view with print support
